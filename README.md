# soul
现在我正在做我的毕业设计，主要是基于KV剪枝的推理加速工作，具体思路为，在DART和SparseVLM的工作基础上，在不引入全量注意力计算的前提下，利用输入文本的语义协同枢轴选择，实现对长上下文多模态数据的压缩处理。
核心设计
层级解耦：浅层保留基本结构，深层关注文字语义。
双重枢轴：结构枢轴 + 语义枢轴协同工作。
动态预算：基于动态熵等指标自动调节压缩率。
